# Getting Started Locally

1. Install dependencies:

   ```bash
   pnpm install
   ```

2. Start the local Server:

   ```bash
   pnpm dev
   ```

3. Open the [Config file](./src/data/resume-data.tsx) and make changes

# Run with Docker

Build the container

```
docker compose build
```

Run the container

```
docker compose up -d
```

Stop the Container

```
docker compose down
```
